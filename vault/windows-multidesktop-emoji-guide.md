# Windows 멀티 데스크탑 이모지 네이밍 가이드

## 목적

신규 PC에서도 기존과 동일한 기준으로 멀티 데스크탑 이름을 빠르게 재구성하기 위한 개인 표준 문서입니다.

## TL;DR

```text
🔐 knox
📚 confluence
📋 jira
🌐 web
🎓 edu
▶️ youtube
🤖 ai
🔥 prj-dotfiles
🚀 prj-B
🌱 prj-C
```

## 네이밍 기준

- 보안/인증: `🔐 knox`
- 문서/지식 관리: `📚 confluence`
- 이슈/작업 트래킹: `📋 jira`
- 웹 탐색/리서치: `🌐 web`
- 학습/교육: `🎓 edu`
- 영상 학습/레퍼런스: `▶️ youtube`
- AI 실험/업무: `🤖 ai`
- 프로젝트 실행/추진: `🔥 prj-dotfiles`
- 핵심 과제/우선순위 높음: `🚀 prj-B`
- 신규 시도/실험성 과제: `🌱 prj-C`

## 대체 이모지 후보

- `knox`: `🛡️`, `🔑`
- `confluence`: `📖`, `🗂️`, `📝`
- `jira`: `🐞`, `🎯`, `🛠️`, `📌`
- `web`: `💻`
- `edu`: `📖`
- `ai`: `🧠`, `🧪`, `🔬`, `✨`
- `prj-dotfiles`: `🧩`, `🛠️`, `📦`
- `prj-B`: `⚡`, `🎯`, `🧠`
- `prj-C`: `🧪`, `📊`, `🧵`

## 적용 체크리스트

1. Windows 가상 데스크탑을 필요한 개수만큼 생성한다.
2. 각 데스크탑 이름을 TL;DR 목록 순서대로 지정한다.
3. 업무 패턴이 바뀌면 대체 이모지 후보에서 1개만 교체하고 나머지는 유지한다.
4. 신규 PC 세팅 시 본 문서를 기준으로 동일하게 복원한다.

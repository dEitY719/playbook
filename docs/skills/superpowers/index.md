인터랙티브 시각 비교는 우측 상단의 **HTML** 버튼(또는 아래 링크)으로 열 수 있습니다.

- **[Claude Code](./index-C/interactive/)** — 대기 분위기 효과와 플로우 다이어그램 중심의 시각화 (`index-C.html`)

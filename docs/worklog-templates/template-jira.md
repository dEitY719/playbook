## 사용 방법

1. 이 템플릿을 복사하여 주간보고 작성
2. 각 섹션에서 [JIRA-KEY]를 실제 Jira 키로 교체
3. 점렛 항목 형식으로 간결하게 작성
4. Work Log 시간 합계가 정확한지 확인
5. Jira 에디터에 복붙 (Markdown 지원)

### 작성 팁
- 각 "완료" 항목은 3개의 점렛(✓ 설명 / ✓ 메트릭 / ✓ 영향)으로 구성
- Time Spent는 `work-log list`나 ~/work_log.txt에서 집계
- Blockers/Risks는 있을 때만 포함
- 참고 링크는 실제 URL로 교체
